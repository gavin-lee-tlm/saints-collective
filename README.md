# The Saints Collective

Static one-page site for **The Saints Collective** — a faith-based 501(c)(3) nonprofit.

## Stack
- Plain HTML/CSS (no build step)
- Hosted on Vercel
- Domain managed via Squarespace → `thesaintscollective.org`

## Structure
- `index.html` — the site (single file)
- `vercel.json` — security headers + clean URLs

## Deployment
Pushes to `main` auto-deploy to production via Vercel's GitHub integration.
